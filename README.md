# INFRACHAT: Natural Language Interface for Manufacturing Infrastructure
InfraChat is an industrial management platform that bridges the gap between complex manufacturing data and human operators. By leveraging Natural Language Processing (NLP) and Predictive Maintenance Intelligence, users can interact with their factory infrastructure using plain English commands.

## Key Features

- Query any machine by Product ID (e.g. L47181)
- 30+ supported natural language commands
- Live dashboard with charts, KPIs and gauges
- Risk scoring — Safe, Elevated, High Warning, Critical
- Fuzzy string matching using Rapidfuzz
- Typing indicator and quick reply chips

## Tech Stack
- Backend: Python / Flask
- Data: Pandas, Rapidfuzz
- Frontend: HTML5, CSS3, JavaScript, Chart.js
- Dataset: Machine Predictive Maintenance Classification (Kaggle)
